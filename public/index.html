Learn to Code by Building One Website Every Week.

You can learn the basics of Html, Css and Ruby on Rails pretty easily, but if you want to develop the practical coding skills you need to build web applications, you need to practice by actually building things...

The problem is that as a beginner who has almost zero experience, without proper guidance, practice becomes an unnecessarily difficult and frustrating process.

When I started learning how to code, I spent months on code academy doing coding tutorials and while the tutorials were nice and I was learning a lot, I didn't really know how to build anything.

It wasn't until I got a job as an entry level developer at a local startup and started shipping code every day that I really started to understand how to actually build web applications.

You'll get this Guided-Practice experience with codeweekend and over the next 4 weeks, you'll learn how to code by building one web application every week.

You'll get a detailed tutorial every week that will show you step by step how to build the "app of the week" and follow along as we build, building it yourself alongside.

In the process, you'll develop the practical coding skills you need to build your own projects.
