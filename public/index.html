Coding for Entrepreneurs! (Rails Edition)

Develop Practical Coding Skills and learn how to build web applications that you can sell to costumers.

Over the next 4 weeks, you're going to learn how build web applications with Ruby on Rails.

You'll get a detailed coding tutorial every week that will show you how to build a complete web application from scratch.

Your instructor (Mike Hibbert) will take you through the process of building the web application and teach you everything you need to complete the web application.

In the process, you'll get a deep understanding of web development and develop the practical coding skills that you need to build your own projects. Even if you have never coded before.

Week 1 - Build a Landing Page (MVP)

To build web applications, there are some basic skills you need to have. HTML, CSS and a basic understanding of Ruby, Git and Heroku. You will develop these basic skills in Week 1 and build a beautiful landing page with HTML and CSS. Then you will deploy the website to Heroku and make it available online, using Ruby, Rack and Github for version control

Week 2 - Build a Blog Engine

In week 2, you'll be introduced to web development and learn how to build web applications with ruby on rails. You will build a simple blog engine that you can use to manage any blog, and deploy it to Heroku again so it is available online. This will be your first Web Application.

Week 3 - Build a URL Shortener

In Week 3, you'll learn some more advanced rails concepts and build a URL shortener like bit.ly.

Week 4 - Saas Application

In the final week, you'll take your coding skills to the next level and learn about things like Gems, Autenciation (login and logout), Associations, API's and More.

You'll build a multi-user to-do list web application (that is robust enough for you to sell). Then you will integrate Stripe (via API) to collect payments from customers and automatically activate their account if the payment is successful.
